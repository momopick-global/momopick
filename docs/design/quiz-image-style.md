# Quiz Image Style — 모모픽

## 스타일 키워드

- 부드러운 일러스트 (사진 X)
- 파스텔톤 + 보라/핑크 액센트 가능
- 표정이 풍부한 캐릭터
- 인물 중심 (옆 / 정면 / 약간의 클로즈업)
- 깔끔한 배경 (복잡한 배경은 메인 인물을 가림)

## 결과 이미지 (가장 중요)

**용도**: 결과 페이지 hero + Kakao Feed 공유 카드 이미지

| 항목 | 권장 |
|---|---|
| 비율 | 4:5 세로형 또는 1:1 정사각형 |
| 크기 | 640×800 또는 800×800 이상 |
| 구도 | 인물 클로즈업 (얼굴 + 상체) |
| 표정 | 결과 타입 감정이 명확 (밀당형: 살짝 도도 / 직진형: 환한 미소 / 눈치형: 살피는 / 과해석형: 골똘) |
| 배경 | 단색 또는 단순 그라데이션 — 인물에서 시선 분산 X |
| 텍스트 | ❌ 절대 넣지 않음 (제목은 별도 텍스트 영역에서) |

## 결과별 감정 차이 가이드

같은 퀴즈 안에서 결과 4개가 명확히 구분되어야 함:

| 톤 | 색 분위기 | 표정 |
|---|---|---|
| 밝음/적극적 | 따뜻한 노랑·핑크 | 환한 미소, 정면 |
| 따뜻함/내향적 | 옅은 라벤더·민트 | 부드러운 미소, 시선 살짝 측면 |
| 사색적/혼자 | 차분한 파랑·그레이 | 골똘, 시선 아래 |
| 도도/시니컬 | 보라·자주 | 옆 보기, 약간 무표정 |

## 썸네일 이미지

**용도**: 카테고리 허브, 홈 레일, 블로그 promo

| 항목 | 권장 |
|---|---|
| 비율 | 1:1 또는 16:9 |
| 크기 | 600×600 또는 960×540 |
| 구도 | 결과 캐릭터 중 가장 대표적인 1명 또는 콜라주 |
| 톤 | 카테고리 색 그라데이션 활용 가능 |

작은 크기(160px 폭 등)로 노출되어도 의미 인식 가능해야 함.

## 시작 이미지

**용도**: 퀴즈 시작 화면 (인트로) — 결과 캐릭터 군집 또는 대표 그래픽

| 항목 | 권장 |
|---|---|
| 비율 | 4:5 또는 1:1 |
| 구도 | 결과 캐릭터 여러 명 콜라주 또는 추상 그래픽 |

## OG 이미지

**용도**: 카카오/페이스북 공유 카드 (페이지 전체)

| 항목 | 권장 |
|---|---|
| 비율 | 1.5:1 (현재 메인 OG) 또는 1.91:1 (페북 표준) |
| 크기 | 1536×1024 (메인) 또는 1200×630 (페북) |
| 내용 | 결과 캐릭터 + 퀴즈 제목 (큰 폰트) |
| 안전 영역 | 중앙 80% 안에 핵심 |

OG 이미지에는 텍스트 OK (페이지 OG는 검색·공유에서 노출되므로 제목 보강).
**결과 이미지에는 텍스트 X** (개별 결과는 텍스트가 페이지 내에서 별도 노출됨).

## 색 사용 가이드 (이미지 안)

- 사이트 액센트인 보라(`#5a67f2`) 또는 핑크(`#ff5b8a` 등) 톤을 일러스트에 일부 사용 가능
- 단, 결과 캐릭터마다 다른 색 분위기 가능 — 단조로움 방지

## 안티 패턴

| ❌ | ✅ |
|---|---|
| 실사 사진 | 일러스트 |
| 이미지 안 한국어 제목 | 텍스트 없음 |
| 어두운 점집 분위기 | 밝고 부드러운 톤 |
| 결과 4개가 비슷한 캐릭터 | 4개가 한눈에 구분 |
| 작은 디테일 (글자 같은 모티프) | 큰 표정·포즈 |
| 무표정 / 정면 응시만 | 다양한 시선·포즈 |
| 16:9 가로형 (모바일 카드에 잘림) | 1:1 또는 4:5 |

## 새 퀴즈 자산 발주 시 가이드

발주서에 포함:
- 비율 1:1 또는 4:5
- 크기 800×800 또는 640×800 이상
- WebP 변환 가능 (혹은 PNG로 받고 변환)
- 텍스트 없음
- 결과 N개 표정/색 분위기 차이 명시
- 사이트 톤 (밝고 부드러운, 보라/핑크 액센트 가능)

## 점검

`node tools/check-quiz-images.mjs`로 자산 누락 검사. 자세한 건 [../setup/local-development.md](../setup/local-development.md).
