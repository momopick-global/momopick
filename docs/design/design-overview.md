# Design Overview — 모모픽

## 브랜드 컨셉

| 키워드 | 톤 |
|---|---|
| 귀엽고 가벼움 | 1~3분 스낵 테스트 — 부담 없이 |
| 부드럽고 밝음 | 라이트 톤 배경 (`#f4f2fb` 옅은 라벤더) + 보라 액센트 (`#5a67f2`) |
| 자기이해 & 공감 | 진단/평가 아님. "그럴 수 있어" 분위기 |
| 공유하기 좋게 | 결과 카드형 비주얼이 SNS에서 매력적 |
| 모바일 중심 | 390px 기준. 데스크톱은 cap된 폭에 중앙 정렬 |

## 디자인 원칙

1. **무겁지 않게** — 의학·심리학 진단처럼 보이지 않기. 결과는 "재미 + 공감"으로 받게.
2. **결과가 보상** — 결과 화면 비주얼이 가장 정성스럽게 (이미지 + 텍스트 + 공유 카드).
3. **공유 동선 단축** — 결과 → Kakao Feed 1탭으로 끝.
4. **로그인 강제 안 함** — 모든 콘텐츠 비로그인 접근 가능, 저장함만 인증 필요.
5. **모바일 우선** — 거의 모든 트래픽이 모바일. 데스크톱은 모바일 폭 cap.

## 색 기본 톤

- 배경: 라이트 라벤더 (`#f4f2fb`)
- 텍스트: 거의 블랙 (`#333333`)
- 액센트: 보라 (`#5a67f2`)
- 카테고리별 그라데이션:
  - 연애 (love): 핑크 → 자주
  - 심리 (mind): 블루 → 퍼플
  - 소셜 (social): 네이비 → 청록
  - 스타일 (style): 크림 → 차콜
  - 재미 (fun): 레드 → 옐로

자세한 컬러는 [color-system.md](./color-system.md).

## 톤 차이 — 다른 서비스와의 비교

| 다른 서비스 | 모모픽 | 차이 |
|---|---|---|
| 유어타로 (가상) | 모모픽 | 모모픽은 타로 컨셉이 아니라 일반 스낵 퀴즈. 더 가벼움. |
| 점집/타로 운영처 | 모모픽 | 신비주의 X. "친구가 보는 가벼운 테스트" 느낌 |
| MBTI 진단 사이트 | 모모픽 | 정확도/진단보다 결과 카피의 공감 위주 |

## 무엇을 피할 것

- ❌ 어두운 점집 분위기 / 신비주의 / 공포 톤
- ❌ "당신은 무조건 ~", "반드시 헤어집니다" 같은 단정·예언
- ❌ 의학적 진단처럼 보이는 표현
- ❌ 사용자를 평가하거나 깎아내리는 결과
- ❌ 무거운 일러스트 (피·다크판타지 등)
- ❌ 자동 재생 + 큰 사운드 영상

## 무엇을 지향할 것

- ✅ "그럴 수 있어, 다 그래" 톤
- ✅ 결과 캐릭터/일러스트는 표정 풍부, 친근
- ✅ 카드형 UI — 한 화면에 하나의 정보 집중
- ✅ 큰 터치 영역 (모바일 엄지)
- ✅ 부드러운 그림자 + 라운드 (현재 `--radius: 20px`)
