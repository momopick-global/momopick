# Content Tone — 모모픽

## 기본 톤

- 친근함 — "친구가 해주는 가벼운 테스트"
- 공감 — "그럴 수 있어"
- 가벼움 — "재미 + 자기이해", 진단·예언 X
- 따뜻함 — 깎아내리거나 평가하지 않음
- 공유하고 싶은 마무리 — 결과를 친구에게 보내고 싶어지게

## 핵심 원칙

| 지향 | 회피 |
|---|---|
| "그럴 때 있죠" | "당신은 반드시 ~합니다" |
| "이런 결과가 나올 수도 있어요" | "이게 운명입니다" |
| "참고용으로 가볍게 봐주세요" | "100% 정확합니다" |
| "재미 + 자기이해 목적" | "임상 심리 진단" |
| 짧고 직관적 | 길고 학술적 |
| 이모지 적당히 (😊 🎭 ✨) | 과도한 이모지 / 점성술 기호 |

## 금지 표현

- ❌ "반드시"
- ❌ "무조건"
- ❌ "당신은 ~한 사람입니다" (단정)
- ❌ "이별이 머지 않았습니다" (예언)
- ❌ "이런 사람은 안 됩니다" (배제)
- ❌ "정확한 진단" / "임상 결과"
- ❌ 성별·나이·외모 비하

## 권장 표현 패턴

- "이런 타입의 사람들은 ~한 경향이 있어요"
- "결과는 가볍게 참고해 주세요"
- "친구에게도 한 번 해보라고 해보세요"
- "최근 그런 적 있지 않으세요?"
- "오늘 한 번 돌아볼 수 있는 기회"

## 결과 카피 톤별 예시

### 적극적 / 활달한 결과
> "직진형이세요! 마음이 가면 망설이지 않는 타입. 그래서 빠르게 사이가 가까워지는 편이에요. 단, 상대가 따라올 속도도 한 번씩 체크해 보세요."

### 신중 / 내향적 결과
> "눈치형이세요. 상대를 잘 살피고 배려하는 사려깊은 타입이에요. 다만 한 번씩은 본인 감정도 솔직하게 표현해 주는 게 관계에 도움이 돼요."

### 사색적 / 혼자 시간 필요
> "독립형이세요. 본인만의 시간이 정말 중요한 타입. 누구보다 자신을 잘 아는 강점이 있어요. 가까운 사람에겐 그 시간이 왜 필요한지 한 마디 공유해 주는 것도 좋아요."

### 도도 / 시니컬
> "관찰형이세요. 한 발 떨어져서 상황을 보는 차분한 타입. 신중함이 강점이지만, 가끔은 마음 가는 대로 한 번 행동해 보세요. 의외의 변화가 있을 수 있어요."

## 면책 톤

결과 페이지 / 정책 페이지에서 다음 톤 유지:

- 본 테스트는 **재미와 자기이해** 목적입니다
- 의학적·임상적 진단이 아니며, 결과는 가볍게 참고해 주세요
- 중요한 결정은 본인의 상황과 감정을 바탕으로 하시기 바랍니다

이미 `src/content/blog/koSamplePosts.ts`의 `mbti-not-diagnosis` 글에 비슷한 톤 확립됨.

## 카테고리별 톤 미세 조정

| 카테고리 | 톤 |
|---|---|
| 연애 (love) | 따뜻 + 살짝 짖궂은 농담. 자기이해 강조 |
| 심리·MBTI | 가벼운 + 재미. 임상 단정 절대 X |
| 소셜·관계 | 부드럽 + 공감. 갈등 해결 톤 |
| 스타일·취향 | 가볍 + 트렌디 |
| 재미 | 밈 + 유머. 혐오·차별 X |

## 공유 텍스트

- 결과 share text는 짧고 공유하고 싶게 (예: "나의 썸 패턴은 밀당형 🎭")
- 과장된 표현은 자제 ("이거 진짜 소름!" 등은 한두 번이면 OK, 남발 X)

## 운영 관점

- AdSense 정책: 차별·혐오·과장·임상 진단 → 거절 사유. 톤 유지는 심사에도 영향.
- Google 정책: medical claims (의학적 주장) 금지. "심리 테스트로 우울증 진단" 같은 표현은 위험.

## 톤 점검 체크리스트 (글 / 결과 발행 전)

- [ ] "반드시", "무조건" 같은 단정 표현 없음
- [ ] 의학·임상 단정 표현 없음
- [ ] 성별·외모·나이 비하 없음
- [ ] 결과가 사용자를 깎아내리지 않음
- [ ] 면책 톤 1줄 포함 (가벼운 참고용)
- [ ] 공유하고 싶게 마무리됨
